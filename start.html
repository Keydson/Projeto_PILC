Demostração do Robô Sparki

O vídeo acima contém uma pequena demostração do que o robô Sparki é capaz de fazer, além do quanto é simples a estrutura do código para programar o Sparki, deixando fácil até mesmo para quem nunca viu programação.

Além de possuir uma estrutura de código simples, é possível também realizar a programção em blocos, tornando a codificação do Sparki ainda mais lúdica.

A programação em blocos nada mais é do que uma metodologia visualmente amigável e com um objetivo prático para o ensino dos conceitos iniciais ligados à programação, desenvolvimento de softwares/aplicativos e lógica de uma forma geral.

É fácil imaginar a infinidade de projetos que este pequeno robô pode realizar.

Teste o Sparki usando o controle remoto. Primeiro, certifique-se de que a pequena aba de plástico de ativação foi removida da parte inferior do controle remoto.

Sparki é pré-programado com um programa de exemplo que vagueia evitando paredes. Você pode interromper este programa pressionando 7 no controle remoto, o que permitirá que você use o controle normalmente, controlando assim o robô manualmente.

5º Passo - Instalando o software:

Exixtem 3 maneiras de programar o seu Sparki, a primeira é através de um software que está disponível para Windows, Linux e Mac, a segunda maneira é pelo próprio navegador, apesar de ainda estar em fase beta, é possível utilizá-la e por fim a programação em blocos, utilizadas geralmente por crianças entre 7-12 anos, porém pode ser utilizada por quem tem dificuldade em entender a lógica de programação e prefere programar de forma mais lúdica, infelizmente este software só está disponível para Windows.
